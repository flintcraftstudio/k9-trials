Type · Body family · Exploration

Inter alternatives

Inter is the current body face. It works, but it's everywhere — Vercel, Linear, Notion, Stripe, half of Y Combinator. K9 Elements deserves something with more identity. Below: eight alternatives, each shown at the system's full type scale alongside an editorial passage and a UI row, so you can read each one in the contexts it'll actually live in. Display (Bebas Neue) and mono (IBM Plex Mono) are unchanged.

Each card: left — H1/H2/H3 + Body L/M/S + Eyebrow at production sizes right — eyebrow → H2 → paragraph → button row, the most common K9E pattern
Geist Recommended
Vercel · Open source · 9 weights

Vercel's in-house grotesk. Same operational, screen-first lineage as Inter (it's openly inspired by it) but with subtly more character — slightly narrower, slightly more contrast, a sturdier "g" and "a." Reads as deliberate engineering rather than default. Pairs cleanly with Bebas Neue's condensed athletic display.

Why this one. Closest to Inter's pragmatic feel, so you don't lose anything operational. Public Sans Strong second
USWDS · US Gov't standard · 9 weights

The official typeface of the US Web Design System. Operational, neutral, slightly warmer than Inter — designed for forms, tables, and dense regulatory text. Matches the "respected coach explaining the rules" tone almost perfectly. Has a faint civic / institutional gravity that suits a sanctioning body.
